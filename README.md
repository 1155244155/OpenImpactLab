# OpenImpactLab
## ADK AI Agent Bot
A clean, intelligent AI assistant built with **Google's Agent Development Kit (ADK)**, Gemini, and built-in web search.

The agent automatically decides when to search the web for up-to-date information and delivers concise, accurate answers.

## Features

- Intelligent tool use (web search only when needed)
- Clean, colorful command-line interface
- Powered by Gemini + Google Search
- Easy to extend with more tools
- No manual search step — fully agentic

## Prerequisites

- Python 3.10+
- Google Gemini API key (free from [Google AI Studio](https://aistudio.google.com/app/apikey))
